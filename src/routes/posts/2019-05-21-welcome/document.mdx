## How does it work


Whenever the user submits the facility request, the manager will get a notification of the request

Manager can click the request and handle the situation

They can also access the status of the laundry machines.

**The facility request is developed by Kertu Koss and here is the Demo(https://laundrycats-facility-request.herokuapp.com/)**


## The tools

- [React](https://reactjs.org) renders the UI
- [Ant Design](https://ant.design/index-cn) UI
- [Firebase](http://firebase.com) real-time data support
- [create-react-app](https://reactjs.org/docs/create-a-new-react-app.html) builds the app's assets and provides the dev server
